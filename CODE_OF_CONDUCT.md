# Code of Conduct

This project follows a lightweight version of the Contributor Covenant.

## Our pledge

Be kind, constructive, and assume good intent. We're all here to build useful
tooling together.

## Standards

- Use welcoming, inclusive language.
- Accept constructive feedback gracefully.
- Focus on what is best for the project and its users.
- Show empathy toward other contributors.

## Unacceptable

- Trolling, harassment, personal attacks.
- Sexualized language or imagery.
- Publishing others' private information without consent.

## Enforcement

Violations can be reported via a private GitHub security advisory or by
contacting the maintainer. Unacceptable behavior may result in being blocked
from the repository.

This code of conduct is adapted in good faith; it is not an exhaustive
legal document.
